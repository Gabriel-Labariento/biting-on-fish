**Biting on Fish** is a multiplayer, real-time **Java dungeon-crawler** featuring  
procedurally structured rooms, enemy AI, boss fights, client-side rendering, and server-side logic.

seven unique dungeon themes filled with enemies, traps, and multi-phase bosses.
